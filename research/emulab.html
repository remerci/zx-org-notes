1.3.1 DOING Bad Response

xmlrpc问题, wiz上记录了一个google论坛的相关问题, 首先看xmlrpc-server是否启动. 查看后发现该服务未启动, 提示未安装mysql-python模块.

  1. 下载MySQL-python-1.2.3.tar.gz
  2. 设置setupposix.py文件中mysqlconfig的位置
  3. python setup.py build
  4. python setup.py install

1.4 TODO free节点经常不成功

主要原因是节点启动时控制网口识别出错. 思路:在节点上再次执行, 查看日志; DeathByDHCP.html

对应脚本是netif-emulab, 执行日志在/etc/emulab或/var/log, 结果在controlif里 (从脚本得到信息); 或执行 kenv boot.netif.name, https://trac.deterlab.net/wiki/Installation/Nodes
